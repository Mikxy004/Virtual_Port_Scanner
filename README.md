A multithreaded port scanner written and built with python and flask

Features:
+ Multithreaded Scanning
+ Network Scanning
+ Flask Web Dashboard
+ Live Scan Progress Bar
+ JSON Report Export
+ Banner Grabbing
+ Service Detection

Run Locally
```bash

cd Port_Scanner
python -m venv venv
source venv/bin/activate
pip install flask
python3 app.py
```
